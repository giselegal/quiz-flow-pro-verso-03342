Teste Manual de Templates

1. Abra Templates

2. Abra o Console (F12)

3. Clique em qualquer template

4. Verifique os logs no console

5. Confirme se navegou para /editor/:id

URLs de teste direto:

  • Editor (sem ID)
  • Editor (com ID test-123)
  • Editor (com ID simulado)

// Cole no console da página /editor-templates:
const templateId = 'quiz-estilo-21-steps';
