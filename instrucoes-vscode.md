# Instruções Passo a Passo para Configurar o Visual Studio Code

## Passo 1: Baixar o Visual Studio Code
- Acesse o site oficial do VS Code em [code.visualstudio.com](https://code.visualstudio.com/).
- Clique em "Download" e selecione a versão apropriada para seu sistema operacional (Windows, macOS ou Linux).
- Execute o instalador e siga as instruções na tela para completar a instalação.

## Passo 2: Instalar Extensões Essenciais
- Abra o VS Code após a instalação.
- Clique no ícone de extensões na barra lateral esquerda (ou pressione `Ctrl+Shift+X` no Windows/Linux, `Cmd+Shift+X` no macOS).
- Pesquise e instale extensões populares, como:
  - Python (para desenvolvimento em Python).
  - GitHub Copilot (para assistência com código).
  - Prettier (para formatação de código).

## Passo 3: Configurar o Ambiente
- Abra as configurações do VS Code pressionando `Ctrl+,` (ou `Cmd+,` no macOS).
- Ajuste preferências como tema, fonte e atalhos de teclado conforme sua preferência.
- Configure o controle de versão Git se necessário: vá para "Terminal" > "Novo Terminal" e execute `git --version` para verificar a instalação.
